# README AND USER MANUAL
You will find how to install and get a quick start on BRiskTyper, which is a genomic pre-processing pipeline for BRisk, following the instructions here: https://github.com/lmc297/BRiskTyper

How to run the BRisk app (with GitHub Desktop + RStudio) 

1. Clone the repository

In GitHub Desktop:

File → Clone Repository

Select: FSL-MQIP/BRisk-app-and-validation

Click Clone

3. Open the app file in RStudio
Open the cloned folder
Go to:
BRisk app/BRisk.R
Open BRisk.R in RStudio

4. Install required packages
All required packages are listed under the “Load packages” section of BRisk.R.

5. Run the app
Click Run App
The Shiny app should automatically launch in your browser.

6. Input files for test runs are saved under: 
For BRiskTyper result, upload files ending with "final_results.csv"
For FastANI result, upload files ending with "cytotoxicity_fastani.csv"
