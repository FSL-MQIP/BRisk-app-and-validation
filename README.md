# README AND USER MANUAL
You will find how to install and get a quick start on BRiskTyper, which is a genomic pre-processing pipeline for BRisk, following the instructions here: https://github.com/lmc297/BRiskTyper

To run BRisk, open file BRisk.R here: https://github.com/FSL-MQIP/BRisk-app-and-validation/blob/main/BRisk%20app/BRisk.R and click Run App.

Input files for test runs are saved under: 
For BRiskTyper result, upload files ending with "final_results.csv"
For FastANI result, upload files ending with "cytotoxicity_fastani.csv"
